About the Game

It's basically a puzzle game that's got this rhythm thing going on

So we grabbed the usual match-3 setup and threw in this visual rhythm element that just... works, you know? Matches trigger wave-like animations, and combos build momentum across the board. It’s not just about clearing tiles — it’s about creating flow.

How to Play

Match, chain, complete

  • Swap Tiles

    Move adjacent elements to create matches of three or more.

  • Create Combos

    Chain matches for stronger effects.

  • Complete Objectives

    Reach goals within limited moves.

  • Use Boosters

    Activate special tiles to clear areas.

  • Advance Levels

    Progress through increasingly complex boards.
